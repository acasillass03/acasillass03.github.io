¿Qué es DHCP?

Definición de DHCP

DHCP (Dynamic Host Configuration Protocol) es un protocolo de red que permite asignar automáticamente direcciones IP y otros parámetros de configuración (como puerta de enlace, máscara de subred o DNS) a los dispositivos que se conectan a una red.

Cómo funciona DHCP

El DHCP funciona mediante un proceso de cuatro pasos principales, conocido como el ciclo DORA (por sus siglas en inglés):

Discover (Descubrir):

El cliente (por ejemplo, tu PC) que se conecta a la red no tiene dirección IP. Envía un mensaje DHCP Discover para buscar un servidor DHCP disponible.

Offer (Oferta):

El servidor DHCP recibe la solicitud y responde con un DHCP Offer, ofreciendo una dirección IP disponible junto con otros datos (máscara, puerta de enlace, DNS, etc.).

Request (Solicitud):

El cliente responde con un DHCP Request, indicando que acepta la dirección ofrecida por ese servidor.

Acknowledge (Confirmación):

Finalmente, el servidor envía un DHCP Acknowledge (ACK) confirmando la asignación. El cliente ya puede usar esa dirección IP durante un tiempo determinado (llamado lease o concesión).
